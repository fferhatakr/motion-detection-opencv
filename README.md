# Motion Detection Project 

Bu proje, Python kullanarak görüntülerdeki hareketleri tespit eden bir uygulamadır.

## Kullanılan Teknolojiler
* **Python 3.11** - Temel programlama dili.
* **OpenCV** - Görüntü işleme ve hareket tespiti için.
* **Dlib** - Yüz ve nesne tanıma yetenekleri için.
* **NumPy** - Matematiksel hesaplamalar için.



##  Kurulum
1. Bu depoyu klonlayın:
   ```bash
   git clone https://github.com/fferhatakr/motion-detection-opencv.git
2.Gerekli kütüphaneleri kurun:
  pip install -r requirements.txt
  
3.Projeyi başlatmak için şu komutu çalıştırın:
  python main.py
